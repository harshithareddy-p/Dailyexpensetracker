# Dailyexpensetracker
💰 Daily Expense Tracker — A Python-based application to record, manage, and track daily expenses. Built to practice programming fundamentals, data handling, and creating a practical real-world finance management tool.
## 📌 About The Project

Daily Expense Tracker is a first-year engineering project developed using C programming.

This console-based application helps users manage their daily expenses by allowing them to add expenses, view recorded expenses, and calculate total spending.

The main purpose of this project was to understand and apply programming fundamentals such as:

- Arrays
- Loops
- Conditional statements
- User input handling
- Data storage
- Basic problem-solving logic

This project represents my early programming journey and helped me build a strong foundation in C programming by creating a practical real-world application.
